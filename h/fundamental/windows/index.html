windows fundamental

  • System Configuration(MSConfig): Configuration utility (MSConfig) is for advanced troubleshooting, and its main purpose is to help diagnose startup issues. 
    The utility has five tabs across the top. Below are the names for each tab. We will briefly cover each tab in this task. 
      General
      Boot
      Services: The Services tab lists all services configured for the system regardless of their state (running or stopped). A service is a special type of application that runs in the background.  
      Startup
      Tool: There is a list of various utilities (tools) in the Tools tab that we can run to configure the operating system further. There is a brief description of each tool to provide some insight into what the tool is for

  • Computer Management (compmgmt): utility has three primary sections: 
      System Tools: >> event view(log all windows events)
      Storage
      Services and Applications.
  • System Information (msinfo32)
  • Resource Monitor (resmon)
