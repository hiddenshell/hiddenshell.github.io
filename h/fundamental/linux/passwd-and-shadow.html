Everything About etc/passwd and etc/shadow File


Crack shadow file users password hash

Create two file: One >> pass.txt which is contain passwd file content
Another one >> shadow.txt which is contain shadow file content.

Now unshadow those files using unshadow pass.txt shadow.txt > passwords.txt
here passwords.txt conatain all information about Users.

Now crack this passwords.txt file with john by this command:
john --wordlist=/usr/share/wordlists/rockyou.txt passwords.txt



Add new user though passwd and shadow file

First make user encrypted password:

openssl passwd -1 -salt [salt value] {password}
or
mkpasswd -m SHA-512 pass

Then, add this line to etc/passwd file.

user2:[enc password]:0:0:,,,:/home/user2:/bin/bash
