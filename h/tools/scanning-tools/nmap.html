Nmap

Nmap is a network scanner created by Gordon Lyon. Nmap is used to discover hosts and services on a computer network by sending packets and analyzing the responses. Nmap provides a number of features for probing computer networks, including host discovery and service and operating system detection


Nmap command line arg

Host discovery:
  • host discovery only: -sn

  • ARP Method: -PR
  • ICMP Method: -PE
  • ICMP Timestamp Method: -PP
  • ICMP MASK Method: -PM
  • TCP SYN Method: -PS // -ps21, -ps23,80
  • TCP ACK Method: -PA *root priv // -pa21, -pa23,80
  • UDP Method: -PU
Normal Port Scans:
  1. No DNS lookup: -n
  2. Reverse-DNS lookup: -R
  3. Port scan: -p[port number] / p[port range]
  4. Scan popular 100 ports: -F **
  5. Skip ping scan: -Pn

  6. Full TCP Scan cmd: -sT
  7. TCP Stealth scan / SYN (half scan): -sS **
  8. ACK flag probe: -sA
  9. os detection scan: -O
  10. UDP scan: -sU
  11. Sent packet per sec: --min-rate [number]
  12. Timing Scan: -T[0-5]
    • paranoid (0): scans one port at a time and waits 5 minutes between sending each probe
    • sneaky (1): is often used during real engagements where stealth is more important.
    • polite (2)
    • normal (3): Normal scan
    • aggressive (4): is often used during CTFs and when learning to scan on practice targets,
    • insane (5): is the fastest
Advanced Port Scans:
  1. Null Scan: -sN
Grab banner & Aggresive scan:
  • For version: -sV **
  • More detail aggresive scan: -A
Firewall Bypass:
  1. Fragment Packet
    • fragment with default size 8: -f
    • fragment with custom size: --mtu [size]
  2. Decoy - spoof with random ip: -D RND:16
  3. Decoy - spoof with specified ip: -D [ip 1],[ip 2],ME...
  4. Spoof Ip: -S [spoof ip] -e [iface]
  5. Spoof MAC: --spoof-mac 0
  6. Check Firewall: --badsum
Vulnability Scanning:
  • Auth check: --script auth
  • Brute-Force: --script brute
  • exploit check: --script exploit
