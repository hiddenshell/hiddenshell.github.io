Nmap

Nmap is a network scanner created by Gordon Lyon. Nmap is used to discover hosts and services on a computer network by sending packets and analyzing the responses. Nmap provides a number of features for probing computer networks, including host discovery and service and operating system detection


Nmap command line arg

1. ping scan: -sn
port scan: -p[port number] / p[port range]
Full Open Scan cmd: -sT
2. skip ping scan: -Pn
scan popular ports: -F
os detection scan: -O
Stealth scan / SYN (half scan): -sS **
3. tcp inverse scann: -sN
4. ACK flag probe: -sA
5. aggresive scan:
  1. For version: -sV **
  2. More detail aggresive scan: -A
UDP scan: -sU

write output
[cmd] > [location]
grepable output: -oG
