Nmap

Nmap is a network scanner created by Gordon Lyon. Nmap is used to discover hosts and services on a computer network by sending packets and analyzing the responses. Nmap provides a number of features for probing computer networks, including host discovery and service and operating system detection


Nmap command line arg

Host discovery:
  • host discovery only: -sn

  • ARP Method: -PR
  • ICMP Method: -PE
  • ICMP Timestamp Method: -PP
  • ICMP MASK Method: -PM
  • TCP SYN Method: -PS // -ps21, -ps23,80
  • TCP ACK Method: -PA *root priv // -pa21, -pa23,80
  • UDP Method: -PU
Normal Port Scans:
  1. No DNS lookup: -n
  2. Reverse-DNS lookup: -R
  3. Port scan: -p[port number] / p[port range]
  4. Scan popular 100 ports: -F **
  5. Skip ping scan: -Pn

  6. Full TCP Scan cmd: -sT
  7. TCP Stealth scan / SYN (half scan): -sS **
  8. UDP scan: -sU
  9. Sent packet per sec: --min-rate [number]
  10. Timing Scan: -T[0-5]
    • paranoid (0): scans one port at a time and waits 5 minutes between sending each probe
    • sneaky (1): is often used during real engagements where stealth is more important.
    • polite (2)
    • normal (3): Normal scan
    • aggressive (4): is often used during CTFs and when learning to scan on practice targets,
    • insane (5): is the fastest
Advanced Port Scans:
  1. Null Scan: -sN
  2. FIN Scan: -sF
  3. Xmas Scan: -sX
  4. TCP ACK Scan: -sA // Discover Firewall
  5. Window Scan: -sW //Discover Firewall
Grab banner & Aggresive scan:
  • For version: -sV ** //--version-light, --version-all, --version-intensity [0-9]
  • OS detection scan: -O
  • Traceroute: --traceroute
  • More detail aggresive scan: -A
Firewall Bypass:
  1. Fragment Packet
    • fragment with default size 8: -f
    • fragment with default size 16: -ff
    • fragment with custom size: --mtu [size x 8]
  2. Decoy - spoof with random ip: -D RND:16
  3. Decoy - spoof with specified ip: -D [ip 1],[ip 2],ME...
  4. Spoof Ip: -S [spoof ip] -e [iface]
  5. Spoof MAC: --spoof-mac 0
  6. Idle/Zombie Scan: -sI [zombie_ip]
  7. Check Firewall: --badsum
Nmap Scripting Engine (NSE)

Script Category Description
auth Authentication related scripts
broadcast Discover hosts by sending broadcast messages
brute Performs brute-force password auditing against logins
default Default scripts, same as -sC
discovery Retrieve accessible information, such as database tables and DNS names
dos Detects servers vulnerable to Denial of Service (DoS)
exploit Attempts to exploit various vulnerable services
external Checks using a third-party service, such as Geoplugin and Virustotal
fuzzer Launch fuzzing attacks
intrusive Intrusive scripts such as brute-force attacks and exploitation
malware Scans for backdoors
safe Safe scripts that wonâ€™t crash the target
version Retrieve service versions
vuln Checks for vulnerabilities or exploit vulnerable services
