Protocol and Services

Protocols

  • Telnet:

    Port: 23

    Example: telnet [IP] , telnet [IP] [PORT]

  • FTP (File Transfer Protocol):

    Port: 21

    Example: telnet [IP] 21

    Example: ftp [IP] [PORT]

  • SSH (Secure Shell):

    Port: 22

    Example: ssh username@IP

    Example: copy file from server - scp user@IP:[server path] [local path]

    Example: copy file from local - scp [local path] user@IP:[server path]
